# Consultor-de-CEP
Projeto simples criado para estudos, onde exploro o consumo de API e a criação de interfaces gráficas com Python. A aplicação permite digitar qualquer CEP e buscar automaticamente as informações, usando a API da AwesomeAPI.
Comecei apenas testando requisições e, por curiosidade, acabei transformando tudo em uma interface funcional usando Tkinter. Foi uma experiência divertida e abriu minha visão sobre como Python é versátil e como pequenas ideias podem virar projetos completos.

✨ Funcionalidades
Busca de informações de CEP via API
Interface gráfica simples e direta feita com Tkinter
Exibição de endereço, estado, bairro e DDD
Versão executável gerada para fácil utilização
🛠️ Tecnologias utilizadas
Python
Tkinter
Requests
AwesomeAPI
🎯 Objetivo

Aprender, experimentar e transformar curiosidade em prática. Esse projeto faz parte do meu processo contínuo de estudos na pós-graduação da PUC Minas e tem me mostrado como as coisas podem ser mais simples do que parecem.
